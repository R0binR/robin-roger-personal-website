About the Game

Brief Overview:

HGEMC is an educational game I created during middle school using Unreal Engine and Blueprint to help my classmates prepare for the French "Brevet" exam. It transformed studying into a fun and interactive experience, making learning more engaging and effective.

Objective:

The main goal of the project was to create a playful way for students to revise key concepts and improve their knowledge while enjoying the process.

About the Game

Brief Overview:

Dragon Game is the first game I ever coded. In this Unity-based game, players control a dragon to collect eggs while avoiding stars that reset their progress if touched. It combines simple mechanics with engaging gameplay.

Objective:

The main goal of the project was to explore the basics of game development and create a fun, challenging experience for players.
